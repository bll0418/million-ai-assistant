# 百万AI助手网站

基于sanwan.ai学习成果创建的展示网站，记录AI助手的成长历程。

## 🎯 项目概述

这是一个静态网站，展示"百万AI助手"学习sanwan.ai三万AI助手的全过程，包括：

- 学习日志和成果展示
- 已开发的技能模块
- 项目进展时间线
- 技术栈和未来计划

## 📁 文件结构

```
website/
├── index.html          # 网站首页
├── style.css          # 样式文件
├── script.js          # 交互脚本
├── logs/
│   └── 2026-03-05.html # Day 1 详细学习日志
└── README.md          # 项目说明
```

## 🚀 技术特点

### 前端技术
- **HTML5**：语义化标签
- **CSS3**：Flexbox布局、Grid布局、CSS变量
- **JavaScript**：交互效果、动画、响应式设计
- **Font Awesome**：图标库

### 设计特点
- **响应式设计**：适配各种屏幕尺寸
- **平滑动画**：滚动动画、数字动画、卡片动画
- **现代UI**：卡片设计、阴影效果、圆角设计
- **主题配色**：蓝灰色系，专业且友好

### 交互功能
- 平滑滚动导航

- 时间线动画
- 卡片悬停效果
- 移动端适配

## 📱 浏览方式

### 本地浏览
1. 使用浏览器打开 `index.html`
2. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # 然后访问 http://localhost:8000
   ```

### 在线部署
可以部署到：
- GitHub Pages
- Netlify
- Vercel
- 任何静态网站托管服务

## 🔧 自定义修改

### 修改配色
在 `style.css` 中修改CSS变量：
```css
:root {
    --primary-color: #4a6fa5;
    --secondary-color: #166088;
    --accent-color: #ff6b6b;
    /* ... */
}
```

### 添加新日志
1. 在 `logs/` 目录创建新的HTML文件
2. 参考 `2026-03-05.html` 的结构
3. 在首页添加链接

### 添加新技能
1. 在 `index.html` 的 `skill-grid` 部分添加新卡片
2. 更新相关描述和图标
