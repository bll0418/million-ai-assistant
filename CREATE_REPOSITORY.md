访问：https://github.com/new

### 步骤2：填写仓库信息
- **仓库名**：`million-ai-assistant`
- **描述**：`百万AI助手网站 - 学习sanwan.ai成果展示`
- **公开仓库**：✅ 选择Public
- **初始化选项**：❌ **不要**勾选：

  - ❌ Add .gitignore
  - ❌ Choose a license

### 步骤3：创建仓库
点击 **Create repository** 按钮

## 🔄 后续命令

### 步骤4：推送代码
```bash
# 在website目录中执行
git push -u origin main
```

### 步骤5：启用GitHub Pages
1. 访问：`https://github.com/bll0418/million-ai-assistant/settings/pages`
2. 分支：选择 `main`
3. 文件夹：选择 `/` (根目录)
4. 点击 **Save**
